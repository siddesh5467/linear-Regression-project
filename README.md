# Linear Regression PRO

This project is a Jupyter Notebook that shows how to use linear regression to predict fuel efficiency (MPG) using the Auto MPG dataset.

## Requirements

- Python 3.x
- Jupyter Notebook or Google Colab
- pandas, numpy, matplotlib, seaborn, scikit-learn, kagglehub

## Usage

1. Open `Linear_Regression_PRO.ipynb` in Jupyter.
2. Run each cell in order.
3. The notebook downloads the dataset, processes it, trains a model, and shows results and plots.

## Data

- Uses the UCI Auto MPG dataset.
- Handles missing values in the `horsepower` column.

## Output

- Trains a linear regression model and reports accuracy metrics.
